# GENESIS

An Artificial Civilization Engine

> Don't build a civilization.
>
> Build the conditions under which civilization becomes inevitable.

Genesis adalah proyek jangka panjang untuk mengeksplorasi bagaimana kehidupan, kecerdasan, masyarakat, dan peradaban dapat muncul dari interaksi sederhana tanpa diprogram secara langsung.
